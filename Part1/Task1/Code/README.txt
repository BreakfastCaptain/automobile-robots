bin2pcd.ipynb: is used to turn frame.pcd.bin to pointcloud.pcd
Dataset: framex.pcd.bin

open3dVis.ipynb: is used to show the content of the image at each stage of the point cloud processing for easy comparison
Dataset: pointcloudx.pcd

DBSCAN.ipynb: generates json files for DBSCAN clustering (json files are generated by pressing D after visualization through open3d)
Dataset: pointcloudx.pcd

K-means.ipynb: generates json files for K-means clustering (json files are generated by pressing D after visualization through open3d)
Dataset: pointcloudx.pcd
